# E-Commerce-Back-End

App that can retrieve, update, add, and delete data through api test platforms such as insomnia

## Technologies include:
-sequelize

-mysql2

-express

-javascript

## Instructions

*log into SQL and create database with schema 

*run node seeds/index.js

*npm start

*make api calls!
